CSS

Cascading Style Sheet

Basic structure

selector{
    property: value;
}

Example

<button>Hello</button>

Change the text color to red

button{
    color: red;
}
<button class="btn">Hello</button>

Change the text color to red

.btn{
    color: red;
}

CSS Selectors

  • Attribute selector
  • Combinators

Reference
  • W3 - MDN

button{ /* Type */
    color: red;
}

.btn{ /* Class */
    color: red;
}

#btn{ /* ID */
    color: red;
}

*{ /* Universal */
    margin: 0;
}
[class="col"]{ /* Attribute */
    margin: 0;
}

ul li{ /* Descendant */
    margin: 0;
}

ul > li{ /* Child */
    margin: 0;
}

li + li{ /* Adjacent sibling */
    margin-left: .5em;
}

Javascript using jQuery

How to include / import jQuery

<!doctype html>
<html lang="en">
<head>
    ...
</head>
<body>
    
    <script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
    <!-- or -->
    <script src="path-to/jquery.js"></script>
</body>
</html>

Why jQuery?

  • DOM manipulation
  • DOM traversing
  • Event handler
  • Effects
  • More...

Basic structure

Before

$(document).ready(function () {
    // code goes here
});

Now

var Site = {
    init: function () {
        // code goes here
    }
};

$(function () {
    Site.init();
});
var Site = {
    init: function () {
        Site.googleMapApi();
